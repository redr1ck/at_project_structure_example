## eng
### To run the tests:
1. Clone the repository.
2. Create and activate a virtual environment (venv):

      ```
      python3 -m venv venv

      # On Windows:
      venv\Scripts\activate

      # On macOS/Linux:
      source venv/bin/activate
      ```
3. Install dependencies: 

      `pip install -r requirements.txt`

4. Run the tests: 

      `pytest -rva tests/test_users.py`
